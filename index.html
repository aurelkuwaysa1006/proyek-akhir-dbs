Web development adalah proses pembuatan dan pengembangan website menggunakan teknologi seperti HTML, CSS, dan JavaScript. HTML itu digunakan untuk menyusun struktur halaman web, CSS itu untuk mengatur atau mempercantik tampilan dan desain, sedangkan JavaScript berfungsi untuk menambahkan interaksi pada website.
